# 数据结构智能学习平台 — 需求规格说明书

## 1. 项目概述

### 1.1 项目背景

数据结构是计算机科学的核心基础课程，但传统教学方式存在以下痛点：
- **抽象概念难以理解**：链表、树、图等数据结构的操作过程缺乏直观展示
- **个性化学习不足**：不同学生的学习路径和薄弱点各不相同，统一教学难以兼顾
- **知识关联薄弱**：数据结构知识点之间存在紧密的依赖与关联关系，传统教材以线性方式组织，割裂了知识体系
- **反馈延迟**：学生在练习中遇到问题时无法即时获得针对性指导

本项目利用**知识工程**技术，构建一个智能化的数据结构学习平台，通过知识图谱组织数据结构领域的知识点，提供可视化演示、智能推荐学习路径、自适应练习等功能。

### 1.2 项目目标

1. 构建数据结构领域的**知识图谱**，包含知识点实体、前置依赖、包含关系、应用关联等语义关系
2. 提供**交互式可视化**，直观展示各类数据结构的操作过程
3. 实现**智能学习路径推荐**，基于知识图谱推理和用户学习状态，推荐个性化学习路径
4. 提供**自适应练习系统**，根据用户的知识掌握程度动态调整题目难度

### 1.3 用户角色

| 角色 | 描述 |
|------|------|
| 学生/学习者 | 使用平台学习数据结构知识、观看可视化演示、完成练习 |
| 教师/管理员 | 管理知识图谱内容、上传题目、查看学生学习数据 |

## 2. 功能需求

### 2.1 知识图谱模块

**FR-KG-01 知识点管理**
- 系统应支持知识点实体的 CRUD 操作
- 每个知识点包含：名称、描述、难度等级、所属类别（线性结构/树形结构/图形结构/排序算法/查找算法等）
- 支持定义知识点之间的语义关系：前置依赖（prerequisite_of）、包含关系（subtopic_of）、相关关系（related_to）、应用关系（applied_in）

**FR-KG-02 知识图谱可视化**
- 以力导向图或层级图形式展示知识图谱全貌
- 支持按类别筛选、按关系类型筛选
- 点击节点展示知识点详情和关联知识卡片
- 高亮显示用户已掌握/未掌握的知识点（绿色/灰色）

**FR-KG-03 知识推理**
- 基于当前用户已掌握的知识点，推理推荐下一步应学习的知识点
- 支持检测知识盲区：用户未掌握的前置依赖知识点
- 生成学习路径（最短路径、推荐路径）

### 2.2 可视化演示模块

**FR-VS-01 线性结构演示**
- 顺序表（ArrayList）：插入、删除、扩容的动态演示
- 链表（LinkedList）：单链表、双向链表、循环链表的节点操作动画
- 栈（Stack）：入栈、出栈的 LIFO 动画演示
- 队列（Queue）：入队、出队的 FIFO 动画演示

**FR-VS-02 树形结构演示**
- 二叉树：构建、前序/中序/后序/层序遍历的逐步动画
- 二叉搜索树（BST）：插入、查找、删除节点的过程动画
- 平衡树（AVL）：旋转操作的逐步演示
- 堆（Heap）：上浮、下沉操作动画
- Huffman 树：构建过程演示

**FR-VS-03 图形结构演示**
- 图的基本表示：邻接矩阵、邻接表的可视化
- 图的遍历：DFS、BFS 的逐步探索动画
- 最短路径：Dijkstra、Floyd 算法执行过程
- 最小生成树：Prim、Kruskal 算法执行过程
- 拓扑排序：AOV 网的排序过程演示

**FR-VS-04 算法演示**
- 排序算法：冒泡、选择、插入、快速、归并、堆排序的比较动画
- 查找算法：顺序查找、二分查找、哈希查找的演示
- 算法复杂度可视化：以柱状图对比不同算法的时间/空间复杂度

**FR-VS-05 通用交互**
- 播放/暂停/单步/重置控制
- 速度调节滑块
- 代码同步高亮（演示过程同步展示对应伪代码/实时代码的执行行）

### 2.3 智能学习模块

**FR-SL-01 学习路径推荐**
- 根据用户目标和当前水平，基于知识图谱生成个性化学习路径
- 路径展示为知识点序列，标明每个知识点的预估学习时间
- 支持推荐"最短路径"、"全面掌握"、"薄弱补强"三种模式

**FR-SL-02 知识掌握评估**
- 通过练习结果计算每个知识点的掌握度（0~100%）
- 采用贝叶斯知识追踪（BKT）模型估计用户的知识状态
- 展示个人知识雷达图和掌握度矩阵

**FR-SL-03 自适应练习**
- 根据用户知识掌握度，从题库中选取适当难度的题目
- 对薄弱知识点增加练习频率（间隔重复策略）
- 题目类型：选择题、填空题、代码补全题、操作模拟题

### 2.4 用户系统模块

**FR-US-01 用户注册与登录**
- 支持邮箱/用户名注册和登录
- 支持 JWT Token 认证

**FR-US-02 个人中心**
- 学习进度概览（已完成/进行中/未开始的知识点）
- 学习时长统计
- 练习正确率统计
- 知识掌握度雷达图

## 3. 非功能需求

### 3.1 性能需求
- 知识图谱可视化加载时间 ≤ 2 秒（100 个节点规模）
- 可视化动画帧率 ≥ 30 FPS
- API 响应时间 ≤ 500ms（P95）

### 3.2 可用性需求
- 界面响应式设计，支持 PC 端和移动端
- 主要功能的操作步骤 ≤ 3 步
- 提供新用户引导流程

### 3.3 安全需求
- 用户密码使用 bcrypt 加密存储
- API 接口使用 JWT 进行身份认证
- 防止 XSS、CSRF、SQL 注入攻击

### 3.4 可扩展性需求
- 知识图谱 Schema 支持灵活扩展新的关系类型
- 可视化组件支持插拔式添加新的数据结构演示
- 题库支持批量导入导出

## 5. 用例描述

### 5.1 核心用例：学习一个数据结构知识点

1. 用户登录后进入学习仪表盘
2. 系统展示知识图谱，高亮推荐学习的知识点
3. 用户点击一个知识点（如"二叉搜索树"）
4. 系统展示知识点详情页：
   - 概念说明文字
   - 交互式可视化演示（插入、查找、删除操作）
   - 相关知识点卡片（前置：二叉树；后置：AVL 树）
5. 用户观看演示并可交互操作
6. 用户完成配套练习
7. 系统更新该知识点的掌握度
8. 知识图谱中该节点颜色从灰色变为绿色

### 5.2 核心用例：教师管理知识图谱

1. 教师登录后进入管理后台
2. 在知识图谱编辑器中添加/编辑知识点
3. 通过拖拽连线定义知识点之间的关系
4. 为知识点关联可视化演示模板和练习题
5. 保存后知识图谱即时更新
